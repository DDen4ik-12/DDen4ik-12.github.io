## 👥 How to contribute? (if you really want to 🤔):
### 📦 Installization for contributing:
1. Install Node >=18 with npm, and Git
2. Make fork of this repository
3. Make local clone of your fork:
```
$ git clone <URL of your fork> <directory name>
```
4. Go to the directory of local clone:
```
$ cd <directory name>
```
5. Install dependencies:
```
$ npm i
```

### 🧪 Testing changes:
1. Install dev dependencies:
```
$ npm i -D
```
2. Start development server:
```
$ npm start
```

### 🌐 Contributing localization:

The `locales` directory contains localization files written in Fluent syntax. If you don't know it, read the [Fluent Syntax Guide](https://projectfluent.org/fluent/guide/) first

When adding translations, follow their order and the order of comments as in other translation files

Some of the names may be untranslatable, for example: `DashBlocks`, `Dash`, `GitHub`, `Scratch`, `Jumper Remake`, names of Dash extensions, names of userscripts, my nickname in original form
